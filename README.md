# clear_up_apks

---

用途：
    遍历某目录的下的三级目录，并查找指定名称文件，复制到指定文件夹
    
    
---

需求的目录结构为

```
android_apk
          hlpq  // 一级目录
              release // 二级目录
                     xxx-xxx-xxx.apk // 三级目录
          zhpq
              release
                     xxx-xxx-xxx.apk
          zlpq
              release
                     xxx-xxx-xxx.apk
          typq
              release
                     xxx-xxx-xxx.apk
          wbpq
              release
                     xxx-xxx-xxx.apk
```
